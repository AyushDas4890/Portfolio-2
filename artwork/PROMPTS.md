# Artwork generation prompts

One abstract artwork per project. Generate with any image model (Midjourney/Imagen/Stitch),
save as `public/artwork/<slug>.jpg`, portrait 3:4, at least 1200×1600.
Until a file exists the site falls back gracefully (monogram card / palette gradient).

Shared style suffix — append to every prompt:
> abstract 3D render, flowing liquid marble, cinematic studio lighting, glossy surfaces,
> deep navy background vignette, high detail, no text, no people, portrait 3:4
